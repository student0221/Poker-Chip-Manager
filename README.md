# 德州扑克筹码管理系统

局域网内的德州扑克筹码管理工具。

## 使用

```bash
# 安装依赖
npm install

# 开发模式（前后端同时启动）
npm run dev

# 生产构建 + 启动
npm run build
npm start
```

## 访问

- **参与者**: `http://<主机IP>:3000/#/`
- **管理后台**: `http://<主机IP>:3000/#/admin`

> 管理路径只有你知道，其他人通过 `#/` 访问参与者界面。

## 流程

1. 管理者打开 `/admin`，设置筹码比例，点击"开始比赛"
2. 参与者打开 `/`，提交姓名、昵称、入场筹码
3. 管理者点击"结束比赛"
4. 管理者输入每位参与者的剩余筹码，点击"执行清算"
5. 系统自动计算排名和盈亏金额

## 技术栈

- React 18 + Vite + Tailwind CSS
- Express + sqlite3
- 响应式设计，支持手机浏览器
